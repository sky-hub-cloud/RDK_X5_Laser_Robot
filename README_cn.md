# RDK_X5_Laser_Robot
# 全自动激光除草机器人

基于 **RDK X5** 与 **STM32F407VET6** 异构双核心架构的全自动激光除草机器人。通过 YOLO26 轻量化模型实时检测田间杂草，驱动二自由度步进云台自动瞄准，并触发高能激光灼烧杂草，全程无需人工干预。

## 📖 项目简介

传统除草依赖化学农药或人工劳作，带来残留污染与人力成本问题。本项目设计了一款全自动激光除草机器人，采用纯物理除草方式，从源头杜绝农药残留，实现绿色农业。

- **异构双主控**：RDK X5 专职 AI 视觉推理，STM32 负责底层运动控制、云台伺服与激光安全逻辑。
- **端到端杂草检测**：部署轻量化 YOLO26 模型，适应户外光照变化，实时输出杂草像素坐标。
- **云台自动追踪**：通过像素坐标-角度映射与 PID 闭环，实现云台双轴稳定瞄准。
- **安全激光控制**：多重互锁保护（目标丢失、过热、障碍），确保仅在定位稳定时触发激光。
- **全自动运行**：上电即自主巡线行驶、识别、瞄准、除草，无需遥控或按键操作。
- **手机实时监控**：通过 Wi-Fi 将 RDK X5 的实时画面传输至手机端，方便远程查看。

## 💻 软件架构

### 1. RDK X5 端（Ubuntu + ROS2）
- **推理节点** (`inference_node.cpp`)：
  - 读取 USB 摄像头图像
  - 使用地平线 BPU 加速推理 YOLO26 模型
  - 输出杂草边界框，提取中心像素坐标
  - 通过串口（`/dev/ttyUSB0`，9600bps）发送 8 字节自定义帧给 STM32
  - 发布带检测框的 ROS2 Image 话题 `/image_with_detections`
- **模型**：YOLO26（Anchor-Free），输入尺寸 640×640，类别数 1（杂草）
- **依赖**：ROS2 Humble，地平线 `hb_dnn` 推理库，OpenCV，cv_bridge

### 2. STM32 端（裸机 HAL）
- **串口中断接收**：解析 8 字节帧（AA 55 为头，X/Y 坐标及置信度）
- **巡线控制**：红外传感器反馈，PID 差速调节（`Set_4_Speed`）
- **云台闭环**：像素坐标 → 角度换算，驱动步进电机（`Emm_V5_Pos_Control`），PID 调节水平/竖直角度
- **激光安全逻辑**：仅当云台定位稳定（误差阈值内）且无故障时，开启激光（PWM 控制 `htim9`）
- **姿态感知**：JY901S 九轴模块（串口 3），用于航向角补偿（可选）
- **OLED 显示**：实时显示角度、状态（调试用）

## 🚀 快速开始

### 1. 硬件连接
- 将摄像头插入 RDK X5 的 USB 口。
- 将 RDK X5 的串口（UART2，PD5/PD6）与 STM32 的 USART2（PD5/PD6）交叉连接（TX ↔ RX）。
- 连接电机驱动器、步进驱动、激光控制线至 STM32 对应 GPIO。
- 供电：锂电池组分别给 RDK X5（5V/2A）、STM32（5V）、电机（12V）、激光（12V）供电。

### 2. 模型部署
- 将训练好的 YOLO26 模型转换为地平线 `.bin` 格式（参考地平线工具链）。
- 将模型文件放置于 RDK X5 新创建的 `/home/sunrise/ros2_ws_10_YOLO26_weed/` 目录（或修改 `inference_node.cpp` 中的 `model_path` 参数，如果直接下载RDK_X5目录，`model_path` 参数则修改为`/home/sunrise/RDK_X5/`）。

### 3. 编译与运行
#### RDK X5 端
```bash
# 进入工作空间
cd ~/ros2_ws_10_YOLO26_weed
colcon build --packages-select inference_node
source install/setup.bash
# 启动节点（参数可调）
ros2 run inference_node inference_node --ros-args -p model_path:=/path/to/model.bin -p camera_id:=0 -p serial_port:=/dev/ttyUSB0 -p baudrate:=9600
